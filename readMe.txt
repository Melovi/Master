Für die Ausführung dieses Programms muss Node.js und darauf aufbauend Meteor installiert sein.

--------------------------------------------------------
auf Linux/Ubuntu:

	sudo apt-get update
	sudo apt-get install nodejs

zusätzlich noch npm:

	sudo apt-get install npm

Dann Meteor:

	curl https://install.meteor.com/ | sh


Nun kann der Meteor benutzt werden. Wenn alles gut gelaufen ist, ist Meteor v.1.0.3+
nun installiert

Jetzt kann in das Root Verzeichnis gewechselt werden und mit dem Befehl

	meteor

die App gestartet werden.

Die App steht nach der Kompilierung im Browser unter localhost:3000 zur Verfügung

VORSICHT!

	> Man ist nicht mit der Melovi Datenbank verbunden sondern hat eine eigene Instanz des Programms auf dem Computer. Daher müssen Videos und Clips neu eingefügt werden!!
